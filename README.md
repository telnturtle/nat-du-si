### 언제부터 낮이고 언제부터 밤인가?

[한국천문연구원 천문우주지식정보 사이트](https://astro.kasi.re.kr/life/pageView/6)에 따르면 2023년 서울 지역의 일출, 일몰 시간은 이러하다.

|          | 일출  | 일몰  | 시민박명 아침 | 저녁  | 항해박명 아침 | 저녁  | 천문박명 아침 | 저녁  |
| -------- | ----- | ----- | ------------- | ----- | ------------- | ----- | ------------- | ----- |
| 1월 5일  | 07:47 | 17:27 | 07:18         | 17:56 | 06:46         | 18:29 | 06:14         | 19:01 |
| 6월 15일 | 05:10 | 19:55 | 04:39         | 20:26 | 04:00         | 21:05 | 03:16         | 21:49 |

일출과 일몰 시간으로 평균을 구하면, 일출은 06:28, 일몰은 18:41 이라고 할 수 있을 것이다.

따라서 이렇게 표기하면 평균적으로 대략 맞을 것이다.

| 밤            | 낮            |
| ------------- | ------------- |
| 06:30 ~ 18:40 | 18:45 ~ 06:25 |

| --------------------- | ------------- |
| 57분 30초 ~ 2분 29초  | 0분           |
| 2분 30초 ~ 7분 29초   | 5분           |
| 7분 30초 ~ 12분 29초  | 10분          |
| 12분 30초 ~ 17분 29초 | 15분          |
| 17분 30초 ~ 22분 29초 | 20분          |
| 22분 30초 ~ 27분 29초 | 25분          |
| 27분 30초 ~ 32분 29초 | 30분          |
| 32분 30초 ~ 37분 29초 | 35분          |
| 37분 30초 ~ 42분 29초 | 40분          |
